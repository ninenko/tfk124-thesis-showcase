# site/ — Фаза 2 (статический фронт → GitHub Pages)

Публикуемый сайт-киоск. Браузер грузит готовый `data/data.json` и рендерит.
❌ Никаких API/CDN/сети в рантайме — всё локально. Полный рецепт — `../WORKFLOW.md` (шаги 7-16).

## Что здесь будет (пока пусто — Фаза 2 не начата)
```
index.html        ← единственная страница: шапка + облако + входы + сетка карточек
app.js            ← логика: рендер, фильтр по тегам/фасетам, киоск-слой
styles.css        ← крупная типографика для зала (≥48pt тело, контраст ≥7:1)
data/data.json    ← собирается build_data.py (Фаза 1)
vendor/           ← ECharts-wordcloud / wordcloud2.js, MiniSearch — локально
fonts/            ← PT Sans или Inter, WOFF2, кириллица — self-host
pdfs/             ← копии исходных ВКР для ссылок «Открыть PDF» (при деплое)
```
